# Story images

Images in this folder are served for the Stories section.

**Optimization (recommended when adding new images):**  
Resize to max width ~1200–1600px and compress (e.g. 80–85% quality) with [Squoosh](https://squoosh.app) or [TinyPNG](https://tinypng.com) before adding. Smaller files load faster with no quality loss for the story layout.
