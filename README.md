# RRI — Resilient Rangelands Initiative

Website for **Resilient Rangelands Initiative (RRI)**, a local organization in Isiolo County, Kenya, committed to strengthening pastoralist communities through locally contextualized solutions.

## Run locally

```sh
npm i
npm run dev
```

## Tech

- Vite, TypeScript, React
- Tailwind CSS, shadcn/ui
- Framer Motion

## Deploy

Build: `npm run build`. Deploy the `dist` folder to any static host (Vercel, Netlify, etc.).
